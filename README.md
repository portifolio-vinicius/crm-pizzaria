# 🍕 CRM Pizzaria - Sistema de Gestão para Pizzaria

CRM completo para gestão de pedidos, clientes, entregas e estoque de uma pizzaria.

## 🛠️ Tecnologias
- Python / Django
- Banco de dados relacional
- Interface administrativa avançada

## 🚀 Como Rodar

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

**Objetivo:** Demonstrar capacidade de construir sistemas práticos e reais para pequenos negócios.
